## 🚀 Getting Started

### Prerequisites
- Flutter SDK 3.x or later
- Android Studio / VS Code
- An emulator or physical device

### Installation

```bash
# Clone the repository
git clone https://github.com/Gbrnd-ux/my-college-tasks.git
cd my-college-tasks

# Install dependencies
flutter pub get

# Run the app
flutter run
